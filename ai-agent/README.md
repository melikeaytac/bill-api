# AI Agent (Assignment 3)

Services:
- frontend (chat UI)
- gateway (all client calls)
- orchestrator (intent parsing + midterm API calls)
- shared (schemas/constants)
- docs

Run (example):
- orchestrator:  cd ai-agent/orchestrator ; npm i ; npm run dev
- gateway:       cd ai-agent/gateway      ; npm i ; npm run dev
- frontend:      cd ai-agent/frontend     ; npm i ; npm run dev
